https://www.lintcode.com/problem/640/ One Edit Distance Given two strings S and T, determine if they are both one edit distance apart. One edit distance means doing one of these operations:
  • insert one character in any position of S
  • delete one character in S
  • change any one character in S to any other character
Example 1 Input: s = "aDb", t = "adb" Output: true
Example 2 Input: s = "ab", t = "ab" Output: false Explanation: s = t, so they aren't one edit distance apart

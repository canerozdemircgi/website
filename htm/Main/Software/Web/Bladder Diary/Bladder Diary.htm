Bladder Diary is a web application that reads a specific formatted text file, in order to track inputs and outputs of human body fluids, across day and night cycles.
It is designed to demonstrate data in a clear, intuitive way and presents through user-friendly, portable interface for both urology doctors and patients.
It generates statistics and shows informative charts, which can provide useful insights, that could be beneficial at diagnosis and treatment planning.
Built as lightweight, client-side solution with no framework or backend to ensure privacy, ease of use and compatibility.

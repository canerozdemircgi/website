Bladder Diary is a web application that reads a specific formatted text file in order to track inputs and outputs of human body fluids. It is designed to demonstrate data in a clear way and presents user-friendly interface for urology doctors and patients.
It generates statistics and shows informative charts which can provide useful insights that could be beneficial at diagnosis and treatment.
It has been built as lightweight client-side solution with html, js, css without any framework or backend dependency to ensure privacy, ease of use and compatibility among diverse operating systems.
